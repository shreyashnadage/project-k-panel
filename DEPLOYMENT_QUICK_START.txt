  AWS Secret Access Key: [your-secret]

================================================================================
TROUBLESHOOTING
================================================================================

ISSUE: "AWS CLI not found"
FIX: Install from https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html

ISSUE: "AWS credentials not configured"
FIX: Run: aws configure

ISSUE: "Docker not running"
FIX: Start Docker application

ISSUE: Deployment fails
FIX: Check CloudFormation events:
     aws cloudformation describe-stack-events --stack-name tally-sync-agent-stack

ISSUE: Instances not healthy
FIX: Check security groups and target group health in AWS Console

ISSUE: API not responding
FIX: Verify ALB DNS and run health check:
     curl http://<ALB_DNS>/health

See DEPLOYMENT_README.md for detailed troubleshooting guide.
